# Details #

Draw a line from one point 0 to point 1.

# Use #

draw\_line(x0,y0,x1,y1,color)
  * Arguments:
    * _uint8\_t_ x0:
      * The x coordinate of point 0.
    * _uint8\_t_ y0:
      * The y coordinate of point 0.
    * _uint8\_t_ x1:
      * The x coordinate of point 1.
    * _uint8\_t_ y1:
      * The y coordinate of point 1.
    * _uint8\_t_ color:
      * The color of the line.
        * WHITE = 1
        * BLACK = 0
        * INVERT = 2
