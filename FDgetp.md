# Details #

Returns the color of the pixel at (x,y).

# Use #
get\_pixel(x,y)
  * Arguments:
    * uint8\_t, x:
      * The x coordinate of the pixel.
    * uint8\_t, y:
      * The y coordinate of the pixel.
    * Returns:
      * unsigned char, The color of the pixel.
        * WHITE = 1
        * BLACK = 0
