# Details #

Sets the color of a pixel at (x,y)

# Use #

set\_pixel(x,y,color)
  * Arguments:
    * uint8\_t, x:
      * The x coordinate of the pixel.
    * uint8\_t, y:
      * The y coordinate of the pixel.
    * uint8\_t, c:
      * The color of the pixel
        * WHITE = 1
        * BLACK = 0
        * INVERT = 2
